leere_zeile

sub leere_zeile

Beim einfügen einer neuen Zeile wird durch das Modul, die vorherige Zeile kopiert.  Diese Funktion fügt eine neue Zeile ein und löscht alle Zellen.  Damit eine leere Zeile in die Tabelle kopiert werden kann.

Parameters

$docReferenz auf Hash mit Pflichtenheft
$bezeichnungName der Tabelle die hinzugefügt wird

Return

$rowleere Zeile wird zurückgegeben
